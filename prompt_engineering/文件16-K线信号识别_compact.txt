【文件定位】K线信号字典——信号棒/入场棒/确认棒等微观信号的定义、识别与交易规则。阶段一逐K信号确认的参考工具，常驻加载。

【核心知识】

一、基本K线角色

1. 信号棒（Signal Bar, SB）
   - 产生交易信号的K线，提示在某方向入场
   - 好信号棒：收盘接近极点、实体较大、尾部短、不超过均长1.5倍
   - 差信号棒：十字星（收盘在中间）、长上下影线、极长实体（过度延伸）
   - 必须后续K线突破信号棒极点才形成有效入场
   - 做多SB：阳线收盘接近高点，入场挂信号棒高点上方1跳动，止损信号棒低点外1跳动
   - 做空SB：阴线收盘接近低点，入场挂信号棒低点下方1跳动，止损信号棒高点外1跳动

2. 入场棒（Entry Bar, EB）
   - 突破信号棒极点、实际触发入场的K线
   - 强EB：大实体收盘接近极点→信号强确认；弱EB：小实体/十字星→风险增加

3. 确认棒（Confirmation Bar）
   - 入场后确认方向的K线（入场棒之后）
   - 强确认：大实体继续创新高/低；弱确认：小实体未创新高/低→交易可能失败

二、外包棒与内包棒

1. 外包棒（Outside Bar, OB）
   - 高点≥前K高点 且 低点≤前K低点，市场扩张
   - 趋势中→延续；区间边界→突破；反转后→确认反转
   - 交易：突破OB极点入场，止损OB另一端外1跳动

2. ioi模式（inside-outside-inside）
   - 三棒组合：IB→OB→IB，突破模式
   - 波动率收缩→扩张→再收缩，预示大幅运动
   - 交易：突破方向入场，止损ioi极点外1跳动，目标ioi高度等幅测量

3. 内包棒（Inside Bar, IB）
   - 高点≤前K高点 且 低点≥前K低点，市场犹豫
   - 单根IB不单独交易（除非在关键位+其他信号支持）

4. ii/iii模式
   - ii：连续2根IB；iii：连续3根IB——波动率极度收缩，必然扩张
   - 趋势中通常沿趋势突破；区间中方向不确定
   - 交易：ii/iii高/低点设止损单，突破顺势入场，止损另一端外1跳动

三、趋势棒与反转棒

1. 趋势棒（Trend Bar, TB）
   - 大实体小尾部，市场一边倒，强动量
   - 标准：实体>均长1.5倍，收盘距极点<20%实体长，尾部极短
   - 可靠性：单根→短暂信号；连续2-3根→强趋势；4根+→极强/微型通道

2. 反转棒（Reversal Bar, RB）
   - 与当前趋势相反的强趋势棒
   - 标准：实体>均长1.5倍，收盘接近其极点
   - 趋势末端+关键位置（支撑/阻力/趋势线/前高低）出现最可靠；趋势中段可能只是回调

四、双棒反转（Two-Bar Reversal, 2BR）

- 两根相邻K线的反转信号：牛市2BR=阴线+阳线反转；熊市2BR=阳线+阴线反转
- 分类：标准2BR（第二根实体≥第一根50%）、吞没2BR（第二根包含第一根）、平底/平顶2BR（形成MDB/MDT）
- 胜率约50-55%，需上下文确认，在关键位置更可靠
- 交易：等2BR收盘完成，入场棒突破2BR极点入场，止损另一端外1跳动，目标2BR高度等幅

五、缺口棒（Gap Bar, GB）与20GB

1. 缺口棒：K线开盘价与EMA之间存在缺口，价格远离均值，趋势极强
   - 多头GB：K线在EMA上方；空头GB：K线在EMA下方
   - GB本身不作入场依据，表示趋势强可顺势（需其他信号支持）
   - 持逆势仓位时GB出现应考虑出场

2. 20GB：连续20根+K线未触EMA，极端强势
   - 不逆势交易20GB趋势
   - 已有顺势仓位→收紧止损
   - 首次回撤到EMA→顺势入场机会
   - 二次回撤到EMA→注意趋势可能结束

六、微双底/微双顶（MDB/MDT）

1. MDB：两根连续K线低点相同或接近（差距<2跳动）
   - 好MDB：第二根收阳、收盘远高于低点、在关键支撑位
   - 交易：突破MDB高点入场，止损MDB低点外1跳动

2. MDT：两根连续K线高点相同或接近（差距<2跳动）
   - 好MDT：第二根收阴、收盘远低于高点、在关键阻力位
   - 交易：突破MDT低点入场，止损MDT高点外1跳动

3. MDB/MDT是大双重顶/底的微观版本，大双重顶/底更可靠

七、尖峰态专属入场信号

⚠️ 尖峰态入场逻辑与其他状态不同——不等待"信号棒+入场棒"确认。

1. 追尖峰信号（SCS）
   - 条件：连续3根+同方向TB，实体>均长1.5倍，几乎无重叠
   - 入场：最新TB收盘价直接入场（不等回撤）
   - 止损：最近回撤K线极点外1跳动（无回撤则最近TB低/高点外1跳动）
   - 目标：尖峰高度等量移动
   - 仅spike_stage="active"时适用；止损>尖峰高度30%时放弃

2. 尖峰回撤入场（SPS）
   - 条件：尖峰中2~4根K线回撤，回撤中出现强趋势棒收盘
   - 入场：回撤信号K线收盘确认后入场
   - 止损：回撤信号棒极点外1跳动
   - 适用：spike_stage="active"或"ending"

3. 最终旗形入场（FFES）
   - 条件：尖峰后第一次回撤形成的旗形被突破
   - 入场：旗形突破信号棒极点外1跳动设止损单
   - 止损：旗形另一端极点外1跳动
   - 适用：spike_stage="ending"或"transitioning"

4. 状态对应
   - active → SCS + SPS
   - ending → SPS + FFES（禁用SCS）
   - transitioning → FFES（其余按通道态规则）

【信号可靠性判断】

可靠性因素（由高到低）：
- 最高：多重信号共振（如MDB+ii+趋势线支撑+EMA支撑）
- 高：信号+关键位置+趋势方向一致
- 中：信号+一般位置
- 低：孤立信号无上下文支持

关键判断维度：位置（支撑/阻力/趋势线/EMA）、上下文（与市场结构一致性）、信号棒大小与收盘、是否有入场棒/确认棒、是否多信号共振。大时间框架信号比小时间框架更可靠。

【例外与陷阱】

1. 信号棒陷阱：孤立信号无上下文；信号棒超均长2倍风险过大；十字星信号买卖犹豫
2. 内包棒陷阱：ii/iii突破方向不确定（尤其区间中）；可能继续收缩不突破；突破后可能假突破回归
3. 外包棒陷阱：OB是扩张信号不等于趋势确认；ioi第三根IB后可能不突破；关键位OB可能是反转非突破
4. 2BR陷阱：胜率仅50-55%；强趋势中常失效；第二根棒<第一根50%则无效
5. 缺口棒陷阱：趋势强≠永远强；20GB后可能继续远离EMA；区间中GB常被填补
6. MDB/MDT陷阱：趋势中常被穿越失效；容差过大则信号无效；可能是更大形态的一部分
7. 通用陷阱：大部分K线是噪音勿过度解读；K线信号必须结合市场结构；追求完美信号不现实
