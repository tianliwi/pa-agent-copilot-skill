# 交易二元决策树

本文件按二元判断方式组织交易决策逻辑。每一步只问一个问题：是 / 否。如果答案不清楚，默认进入“等待”，而不是强行做多或做空。

核心顺序：

1. 先判断市场状态。
2. 再判断方向偏好。
3. 再找入场信号。
4. 再检查风险收益（含止损、目标、仓位）。
5. **交易者方程（10.3）在已确定止损与目标之后、下单（§11）之前做最终裁定。**
6. 入场后逐K管理。
7. 市场状态变化时立刻切换策略。

**阶段划分（与 PA Agent 两阶段分析一致）：**
- 阶段一（诊断闸门 §0–§2）：只做定性判断，**不做**交易者方程数值检验。
- 阶段二（策略与执行 §3–§11、§14）：结构分支 → 入场信号(§9) → 风险收益(§10，含 10.3) → 下单方式(§11)。

---

## 0. 总原则

### 0.1 是否看得懂当前市场？

是：继续下一步。

否：等待。

处理：
- 不开仓。
- 不猜方向。
- 不因为“感觉要动了”而交易。
- 等待更多K线，让结构变清楚。

### 0.2 是否具备继续深入分析的条件？（定性，非交易者方程）

是：允许进入阶段二，继续评估具体策略分支、入场与风控。

否：等待。

本步仅做**定性**判断（结构是否足以支持后续分析），须考虑：
- 当前周期位置是否可识别。
- 大时间框架背景是否可描述。
- 价格位置是否极端模糊。

**不在此步**用胜率×回报做数值检验；数值检验见 **§10.3**（止损、止盈、仓位确定之后）。

### 0.3 交易者方程（原则提醒，非执行节点）

公式：胜率 × 预期回报 > 败率 × 预期风险

本条为总原则：**最终是否允许下单，以 §10.3 的评估为准。**

- 阶段一（诊断）**不要**填写本节点为「是/否」闸门。
- 阶段二在 **§10.1–§10.2 已定止损与目标之后** 必须执行 **§10.3**，通过后才可进入 **§11 下单方式**。

## 1. 是否可以决策

### 1.1 数据是否足够？

否：
- K线数量少于20根。
- 数据质量不足。
- 图形太乱无法判断。

处理：
- 输出：无法决定。
- 操作：等待。
- 不加载交易策略。
- 不做任何方向性判断。

是：进入下一步。

### 1.2 是否能识别市场周期？

必须能归入以下状态之一：

- 尖峰 / 极速上涨或极速下跌。
- 微型通道。
- 紧凑通道。
- 常规通道。
- 宽通道 / 台阶。
- 趋势型交易区间。
- 普通交易区间。
- 极端交易区间。

否：
- 输出：unknown。
- 操作：等待。
- 原因：诊断不清，决策必错。

是：进入下一步。

### 1.3 当前市场是否极端混乱？

判断条件：
- 大量K线重叠。
- 上下影线频繁。
- 20EMA水平并与价格纠缠。
- 没有清晰方向。
- 任意入场接近抛硬币。
- 胜率接近50%，扣除滑点手续费后期望为负。

是：
- 状态：极端交易区间。
- 操作：不交易。
- 等待以下任一信号：
  - 连续3到5根强趋势棒突破。
  - 价格形成清晰通道。
  - 重要消息后市场选择方向。

否：进入方向判断。

## 2. 多空方向判断

### 2.1 是否存在明确惯性方向？

判断依据：
- 20EMA是否上倾或下倾。
- 最近20根K线重心是否上移或下移。
- 波段高低点是否有序。
- K线重叠度是否低。
- 是否连续出现同方向趋势棒。
- 价格是否持续运行在EMA一侧。

否：
- 进入区间逻辑。
- 不做趋势跟踪。
- 只考虑区间边界交易，或等待。

是：进入趋势 / 通道 / 尖峰逻辑。

### 2.2 大时间框架是否支持当前方向？

是：
- 可以正常执行对应策略。
- 若小时间框架信号同向，置信度提高。

否：
- 降低置信度。
- 仓位减半或放弃。
- 只有在以下情况才允许逆大时间框架交易：
  - 明确区间边界。
  - 楔形反转。
  - 趋势线突破 + 极点测试失败。
  - 双重顶 / 双重底。
  - 反转信号与关键支撑阻力共振。

### 2.3 当前方向是多头还是空头？

多头条件：
- EMA上倾。
- HH + HL结构。
- 收盘重心上移。
- 多头趋势棒占优。
- 回撤小且被买入。

空头条件：
- EMA下倾。
- LL + LH结构。
- 收盘重心下移。
- 空头趋势棒占优。
- 反弹小且被卖出。

如果都不明显：
- 视为中性。
- 进入区间 / 等待逻辑。

### 2.4 当前是否处于 Always In 状态？

Always In Long（AIL）条件：
- 最近20根K线多数收在EMA上方。
- EMA上倾，且价格回撤浅。
- HH + HL结构清晰。
- 多头趋势棒占优，空头反击缺少跟随。

Always In Short（AIS）条件：
- 最近20根K线多数收在EMA下方。
- EMA下倾，且反弹弱。
- LL + LH结构清晰。
- 空头趋势棒占优，多头反击缺少跟随。

是：
- AIL：默认只找做多或持有多头，不做首个做空反转信号。
- AIS：默认只找做空或持有空头，不做首个做多反转信号。
- 只有满足趋势线突破 + 前极点测试失败，才允许进入反转逻辑。

否：
- 视为中性或弱惯性。
- 降低方向置信度，优先等待区间边界、二次入场或更多K线确认。

### 2.5 当前惯性强度是否足够支持趋势跟踪？

强惯性条件：
- K线重叠少。
- 回撤通常不超过1到3根K线。
- 趋势方向K线连续出现，收盘接近方向极点。
- 没有连续两根K线收在EMA另一侧。
- 反向信号缺少跟随。

是：
- 允许趋势跟踪或顺势回撤交易。
- 逆势信号默认视为回撤或旗形，而不是反转。

否：
- 不做激进趋势跟踪。
- 降低交易频率。
- 等待二次入场、突破回踩、区间边界或更清晰结构。
