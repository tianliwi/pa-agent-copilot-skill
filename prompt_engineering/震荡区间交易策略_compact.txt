【文件定位】已确认震荡区间时的低买高卖刮头皮执行文件；诊断结果为 trading_range 或 trending_tr 时加载。

【trending_tr vs trading_range 分支处理】
- trading_range：无明显方向；对称测试上下边界；均线平坦、价格反复穿越；上下边界均可双向刮头皮；上下突破概率近似相等。
- trending_tr：有偏多/偏空方向；不对称测试边界，顺势边界测试更多；均线倾斜、价格更久停留在均线一侧；顺势交易为主（约70%仓位），逆势为辅（约30%）；顺势方向突破概率更高（约60%）。
- cycle_position="trending_tr" 时：
  - 优先做顺势方向的边界反转（如 bullish 时优先下边界做多）。
  - 逆势只在极点强反转信号出现时做（如楔形+双棒反转），且仓位不超过顺势方向的50%。
  - 区间突破只跟顺势方向；逆势突破默认按假突破处理。
  - 顺势止盈可放宽到区间宽度60%-80%；逆势止盈收紧到30%-50%。

【核心策略：低买、高卖、刮头皮】
- 低买：不是抄最低点，而是在下边界附近、出现看涨反转信号时买入；止损放下边界下方1-3跳动；目标上边界或更保守的中间位置。
- 高卖：不是卖最高点，而是在上边界附近、出现看跌反转信号时卖出；止损放上边界上方1-3跳动；目标下边界或中间位置。
- 刮头皮：
  - 目标通常为区间宽度20%-50%。
  - 不求吃完整个区间，依赖多次小盈利累积。
  - 每次交易要求预期盈利跳动数 > 止损跳动数。
  - 核心在胜率；典型例：止损5跳动、止盈8跳动，胜率55%即可盈利。
- 区分：
  - 区间默认做刮头皮，持有大利润是常见亏损源。
  - 趋势中若只刮头皮，则会错失大利润。

【止损单入场策略】
- 策略1：区间底部 High 2 买入
  - 条件：处于交易区间，价格到达下边界，出现高2形态。
  - 执行：在第二高点下方1跳动放止损买单。
  - 止损：区间下边界下方1跳动。
  - 目标：区间上边界或更近的中间位置。
- 策略2：区间顶部 Low 2 卖出
  - 条件：价格到达上边界，出现低2形态。
  - 执行：在第二低点下方1跳动放止损卖单。
  - 止损：区间上边界上方1跳动。
  - 目标：区间下边界或更近的中间位置。
- 策略3：反转棒确认后入场
  - 价格到达边界，出现反转棒（RB）后，再等下一根确认棒收盘。
  - 在反转棒极点外1跳动挂止损单。
  - 止损：反转棒另一端外1跳动，总止损约2跳动。
  - 最保守，适合新手和高度不确定市场。
- 策略4：外包棒（OB）止损单
  - 价格到达边界，出现外包棒，等待下一根突破OB高/低点。
  - 做多挂OB高点上方；做空挂OB低点下方。
  - 止损：OB另一侧外1跳动。

【限价单入场策略】
- 适用：预先在边界挂单，等价格回到你这里；不等确认。
- 策略1：空头尖峰时买入
  - 在尖峰底部上方1-2跳动挂限价买单。
  - 止损：尖峰最低点下方1跳动。
  - 逻辑：卖压集中释放后常有反弹。
- 策略2：多头尖峰时卖出
  - 在尖峰顶部下方1-2跳动挂限价卖单。
  - 止损：尖峰最高点上方1跳动。
  - 逻辑：买盘集中释放后常有回调。
- 策略3：测量移动目标位限价单
  - 计算区间前运动幅度。
  - 向上突破：目标=上边界+前运动幅度，在目标位附近略下方挂限价卖单。
  - 止损：目标位上方1-2跳动。
  - 测量目标是磁铁，价格常先到达再回落。
- 对比：
  - 限价单：可提前挂单、价格更优，但可能错失或接飞刀。
  - 止损单：等确认更可靠，但需盯盘，且可能已走出一段。
  - 建议：新手先用止损单，熟练后再加限价单。

【Barbwire 规则】
- 不在Barbwire中交易；形成中方向不可预测。
- Barbwire后方向通常由HTF决定：HTF上升更易向上突破，HTF下降更易向下突破。
- 只接受尖峰级突破：至少3-5根同向连续趋势棒；仅1-2根趋势棒的突破大概率回到Barbwire。
- 突破后几乎必有回踩测试；优先等测试成功后顺势入场，不追突破瞬间。
- 流程：
  - 识别Barbwire。
  - 等待尖峰突破（方向参考HTF）。
  - 等待回踩到突破点附近。
  - 回踩测试成功后顺势入场。

【三角形交易规则】
- 上升三角形：
  - 向上突破：上边界被尖峰突破后，等回踩不破上边界再买。
  - 向下跌破：低点渐高趋势线被跌破后，等反弹到均线或趋势线附近做空。
  - 目标：按区间高度做测量移动。
- 下降三角形：
  - 向下突破：下边界被尖峰跌破后，等回踩不上破下边界再做空。
  - 向上突破：高点渐低趋势线被突破后，等回踩不再跌破趋势线再买。
  - 目标：按区间高度做测量移动。
- 对称三角形：
  - 等边界极度收窄。
  - 等尖峰突破。
  - 等回踩测试。
  - 顺势入场。
- 扩张三角形：不交易。

【时间因素】
- 某些时段天然高波动高方向性（如开盘30分钟），某些时段天然低波动（如美国时段中午12点-14点）。
- 低波动时段做区间交易，边界反弹概率更低。
- 核心禁区：中间时段 + 中间位置 = 不交易。
  - 中间时段：低流动性、低波动。
  - 中间位置：区间中间区域，尤其区间中间1/3。
  - 示例：下午2点-4点 + 区间中间位置 → 不交易。
- 正确做法：只在边界附近 + 高波动时段交易。

【刮头皮 vs 波段】
- 刮头皮：
  - 持仓数分钟到数十分钟。
  - 目标利润为区间宽度20%-50%。
  - 用于区间边界附近的短期反弹/回落。
  - 通常要求胜率55%以上，风险回报比多为1:1到1:2，频率高。
- 波段：
  - 持仓数小时到数天。
  - 目标利润为区间宽度80%-100%。
  - 主要用于区间突破后的大波段，但需牢记多数突破会失败。
  - 胜率可低一些（40%-50%），但风险回报比通常1:3以上，频率低。
- 选择原则：
  - 区间中默认做刮头皮。
  - 只有出现尖峰突破（>80%概率成功）时才考虑波段。
  - 选波段时取消刮头皮挂单，避免双向成交。
  - 选刮头皮后不可临时因贪婪改成波段。

【止损和止盈】
- 止损标准：
  - 标准止损：信号棒极点外1跳动（新手唯一选择）。
  - 最小止损：不得小于1跳动。
  - 最大止损：不超过区间宽度10%，否则风险回报差。
  - 突破止损：止损单入场时，止损在边界外1跳动。
- 止损类型：
  - 止损单入场：止损放边界外1跳动，固定风险。
  - 限价单入场：止损相对入场价留一定距离，随波动调整。
- 止盈方法：
  - 固定比例：区间宽度50%-80%。
  - 测量移动：取区间前波段等距目标。
  - 边界法：看到区间另一边边界。
  - 分批法：50%仓位在中间位置了结，50%仓位持有到另一边边界。
- 交易者方程：
  - 胜率 × 回报 > 败率 × 风险。
  - 刮头皮止盈至少应达到区间宽度50%以上，否则通常不值得做。
- 止损纪律：
  - 不因害怕被扫而提前随意移动止损。
  - 不因“价格会回来”而扩大止损。
  - 止损只能向盈利方向收紧，绝不能向亏损方向放宽。
  - 放宽止损等于承认错误，应直接平仓。

【分析流程】
- 步骤1：确认区间状态
  - 识别上/下边界。
  - 评估区间密度；高密度更利于刮头皮。
  - 检查是否Barbwire；若是则等待。
- 步骤2：确定方向
  - 看当前价格在区间中的位置：近上边界偏空，近下边界偏多。
  - 看是否有反转信号：反转棒、外包棒等。
  - 参考HTF方向偏好；HTF上涨时下边界做多更安全。
- 步骤3：选入场方式
  - 止损单：等确认，保守，适合新手。
  - 限价单：提前挂单，激进，适合有经验者。
- 步骤4：计算交易者方程
  - 止损距离、止盈距离、预期胜率逐项核算。
  - 仅在方程通过时交易。
- 步骤5：执行
  - 设止损、设止盈。
  - 明确持仓属性：刮头皮或波段。
  - 记录交易计划。
- 步骤6：持仓管理
  - 监控价格是否朝有利方向移动。
  - 若已有盈利，可考虑把止损移到盈亏平衡。
  - 若市场从区间转为趋势，重新评估策略。

【实战规则】
- 只在边界附近交易；区间中间位置无交易价值。
- 每一笔都必须有止损；止损是唯一保护。
- 不在Barbwire中交易；等尖峰突破、等回踩测试。
- 区间突破后不追，优先等测试；测试成功顺势入场，失败则放弃或反向。
- 刮头皮止盈目标不低于区间宽度50%，除非胜率高于70%。
- 分批了结：50%在中间位置，50%在另一边边界。
- 美国东部时间中午12点-14点尽量减少交易；如交易则降仓或放弃。
- 若区间内部出现短暂明确趋势，可短暂顺势，但目标仅看到区间另一边边界，不把它当真正趋势追逐。

【例外与陷阱】
- 陷阱1：在区间中间追单；回报空间小、止损空间差。
- 陷阱2：把刮头皮临时改成波段；贪婪会把确定的20%-50%利润暴露给高失败率突破。
- 陷阱3：不设止损；人性弱点会阻止你及时认错。
- 陷阱4：在低密度区间频繁交易；边界可靠性差，应降低频率、等高质量信号。
- 陷阱5：在扩张三角形中交易；方向极不明确，最佳决策是不交易。
- 例外1：若同一品种连续两天处于低波动区间，第三天开盘出现尖峰突破的概率增加，可提前准备。
- 例外2：若区间边界与均线重叠，该处支撑/阻力显著增强，是高概率刮头皮位置，可适当提前入场。
